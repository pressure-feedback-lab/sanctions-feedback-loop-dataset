# Sanctions Feedback Loop Dataset

A structural observation dataset for analyzing
nonlinear feedback loops generated by sanctions,
pressure systems, and constrained connectivity.

This repository focuses on:

- logistics disruption
- insurance friction
- payment fragmentation
- local adaptation
- delayed structural response
- recursive political/economic reactions

The purpose is NOT:

- moral judgment
- geopolitical advocacy
- predictive certainty
- behavioral optimization

Instead, this repository treats sanctions as:

> recursive structural pressure
> interacting with complex adaptive systems.

---

# Observation Perspective

This dataset assumes that:

- pressure does not propagate linearly
- systems react with delay
- silence itself can become a signal
- local adaptation reshapes outcomes
- constrained connectivity produces nonlinear behavior

The repository is intended as:

- an observation protocol
- a structural mapping layer
- a phase-oriented analytical scaffold

rather than a deterministic model.
